# Basics of Computer Networking

## Introduction

A **computer network** is a system that connects two or more devices so they can exchange information and share resources. These connected devices communicate using standard networking rules called protocols, allowing users to transfer files, access the internet, share printers, and use cloud-based services.

Modern computer networks are found everywhere, from homes and schools to businesses and large data centers. Whether you send an email, stream a video, or browse a website, you are using a computer network.

### Key Components

- **Nodes** are devices such as computers, laptops, smartphones, servers, and printers that participate in communication.
- **Networking devices** like routers and switches help move data between connected devices.
- **Transmission media** carries data from one device to another through wired or wireless connections.
- **Communication protocols** ensure that data is exchanged accurately and reliably.

# Working of a Computer Network

A computer network transfers information by following a sequence of communication steps. Every device connected to the network works together to deliver data from the sender to the intended receiver.

## How a Computer Network Works

- Every device connected to the network is assigned a unique address for identification.
- Devices are connected using wired technologies such as Ethernet cables or optical fiber, or wireless technologies such as Wi-Fi.
- When a user sends data, it is divided into smaller units called packets.
- Each packet contains information about its source and destination.
- Network protocols define how packets are created, transmitted, received, and verified.
- Switches and routers examine the destination information and forward packets through the most suitable path.
- At the receiving device, the packets are reassembled into their original form.
- Security technologies such as firewalls and authentication systems help protect the network from unauthorized access.

# Types of Computer Network Architecture

Computer networks are generally designed using one of the following architectures.

## 1. Client-Server Architecture

In this architecture, one or more powerful computers called **servers** provide services to multiple client devices. Clients request resources, and servers process those requests before sending responses.

### Characteristics

- Centralized management of resources.
- Better security and access control.
- Suitable for schools, organizations, and enterprises.
- Easier backup and maintenance.

## 2. Peer-to-Peer (P2P) Architecture

In a Peer-to-Peer network, every device can function as both a client and a server. Devices communicate directly with one another without depending on a central server.

### Characteristics

- Simple to configure.
- Low implementation cost.
- Suitable for small networks.
- Resource sharing occurs directly between connected devices.

# Network Devices

Network devices help establish communication between computers and ensure that data reaches the correct destination efficiently.

## 1. Router

A router connects multiple networks and forwards data between them using IP addresses.

### Functions

- Connects local networks to the Internet.
- Chooses the most efficient route for data.
- Transfers packets between different networks.

## 2. Switch

A switch connects devices within the same local network and forwards data only to the intended destination.

### Functions

- Connects computers, servers, and printers.
- Reduces unnecessary network traffic.
- Improves communication speed within a LAN.

## 3. Hub

A hub is a basic networking device that broadcasts incoming data to every connected device.

### Functions

- Connects multiple devices.
- Sends data to all ports.
- Suitable only for very small or legacy networks.

## 4. Bridge

A bridge joins two network segments and filters traffic between them.

### Functions

- Reduces unnecessary data transmission.
- Improves network performance.
- Uses MAC addresses for forwarding decisions.

## 5. Gateway

A gateway enables communication between networks that use different communication protocols.

### Functions

- Converts data between different protocols.
- Connects dissimilar networks.
- Commonly links private networks with external systems.

## 6. Access Point (AP)

An Access Point provides wireless connectivity to devices within a network.

### Functions

- Extends wired networks into wireless networks.
- Allows smartphones, laptops, and tablets to connect through Wi-Fi.
- Increases wireless coverage.

## 7. Modem

A modem enables communication between a user's network and an Internet Service Provider (ISP).

### Functions

- Converts digital signals to transmission signals and vice versa.
- Establishes an internet connection.
- Used in homes and offices.

## 8. Firewall

A firewall protects a network by monitoring incoming and outgoing traffic according to predefined security rules.

### Functions

- Prevents unauthorized access.
- Filters suspicious traffic.
- Improves overall network security.

# Goals and Uses of Computer Networks

Computer networks are designed to improve communication, resource utilization, and overall system efficiency.

## Resource Sharing

Users can share hardware devices, software applications, and files without maintaining separate resources for every computer.

## Internet and Cloud Access

Networks provide access to websites, cloud storage, online collaboration tools, and various internet-based services.

## Cost Efficiency

Organizations reduce hardware, software, and maintenance costs by sharing common resources across multiple users.

## Reliability and Availability

Modern networks provide backup communication paths and redundant systems to minimize downtime.

## Scalability

Additional devices, users, and services can be added without redesigning the entire network.

## Security and Management

Authentication, encryption, monitoring, and access control mechanisms help protect data and network resources.

# Summary

Computer networking enables multiple devices to communicate, share resources, and access services efficiently. A network consists of connected devices, communication media, networking hardware, and standardized protocols that work together to transfer data securely. Understanding network architectures, common networking devices, and the primary goals of computer networks provides a strong foundation for learning advanced networking concepts such as routing, switching, network security, cloud computing, and distributed systems.
