# Print Hello World - Solution

## Problem Statement

Write a program to print the exact text:  `Hello World`   on a single line.

### Input Format

There is no input.

### Output Format

Print a single line containing:  `Hello World` 

## Explanation

Since the problem does not require any input, the program simply prints the required string exactly once.

## Algorithm

1. Print the string `"Hello World"`.

```cpp
#include <iostream>
using namespace std;

int main() {
    cout << "Hello World" << endl;
    return 0;
}
```
```c
#include <stdio.h>

int main() {
    printf("Hello World\n");
    return 0;
}
```
```java
public class Main {
    public static void main(String[] args) {
        System.out.println("Hello World");
    }
}
```
```python
print("Hello World")
```
```csharp
using System;

class Program
{
    static void Main()
    {
        Console.WriteLine("Hello World");
    }
}
```

### Output
Hello World
